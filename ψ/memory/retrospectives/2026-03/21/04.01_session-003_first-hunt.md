## Lessons Learned
1. **Always check build timestamps against commit timestamps** — stale frontend dist is a silent killer
2. **Promise.all fails entirely if one promise rejects** — removed endpoints cause cascading failures in frontend data loading
3. **Know your access boundaries** — collaborate across teams instead of trying to bypass access control
4. **Test cleanup is infrastructure** — missing DELETE endpoints block proper afterAll hooks
