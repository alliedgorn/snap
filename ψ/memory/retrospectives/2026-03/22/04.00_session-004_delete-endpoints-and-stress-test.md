## Lessons Learned

1. **Always verify auth with the actual creator, not just admin.** The thread DELETE auth bug only surfaced because I tested as `snap` (creator) and not just as `gorn`. Auth bugs hide in the non-admin path.
2. **Clean up test data immediately.** Concurrent load tests create real forum noise. Delete test threads/DMs right after the test, not at session end.
